Apa itu simplidots?
SimpliDots adalah layanan berbasis cloud yang bisa membantu sebuah perusahaan mengawasi stok barang di gudang, hingga memantau proses distribusi. Dengan aplikasi yang mereka buat, kamu bisa melacak keberadaan para supir yang mengantar barang, serta mengetahui secara pasti berapa jumlah barang yang mereka kirimkan.
Mengapa harus menggunakan aplikasi Simplidots?
SimpliDots menghadirkan layanan manajemen keuangan serta pelaporan yang bisa membantu kamu menentukan strategi perusahaan di masa depan.

MASTER DATA

Produk Utama (Product Master)
Bagaimana menampilkan produk yang ada?
    Menampilkan produk utama (produk master)
  1. Pertama pilih menu master data → product → product master

  2. Data dari produk utama(produk master)

    Dari gambar diatas, kita dapat melihat produk-produk yang ada serta data dari produk.


Bagaimana menambah/membuat produk baru?
    Menambah/membuat produk baru
    Semakin banyak permintaan konsumer, semakin banyak pula produk-produk baru yang dibuat untuk memenuhi permintaan. Data dari semua produk harus disimpan dan dicatat.
    Cara untuk menyimpan data dari produk baru, ialah :
  1. Pilih tombol tambah (+)yang berwarna biru , setelah itu akan tampil form untuk menambah produk baru.


  2. Isilah kolom yang ada sesuai dengan data dari produk, kolom yang berwarna kuning merupakan kolom yang wajib* untuk diisi, dan kolom yang berwarna putih tidak wajib untuk diisi.
    *jika tidak diisi, akan ada allert(peringatan) untuk mengisi kolom

  3. Setelah form diisi dan data sudah benar, tahapan selanjutnya ialah menyimpan data.
    Untuk menyimpan data ada dua pilihan :

    Pertama, tombol "Save" untuk menyimpan data
    Kedua, tombol "Save&New untuk menyimpan data dan menampilkan form baru

  4. Data dari produk yang baru, telah selesai dibuat.
Bagaimana mengedit data produk yang sudah ada?
    Mengubah/mengedit data produk
    Data(produk) yang ada pasti mengalami perubahan, untuk itu data harus diubah/di edit dengan data yang terbaru.
    Maka untuk mengubah data :
  1. Pertama, klik pada code produk (tulisan berwara orange)

  2. Rincian data(data sebelumnya) dari produk.

  3. Isilah kolom, dengan data terbaru (data ter-update)
    Jika data sudah pasti, pilih/klik tombol "Save" untuk menyimpan data terbaru.
Bagaimana menghapus data produk?
    Menghapus data produk
    Semakin hari semakin banyak produk baru, kemunculan produk yang baru dapat menggantikan fungsi/tugas dari produk lama. Produk lama menjadi tidak berguna dan tidak terpakai lagi, maka produk lama akan dihapus.
    Untuk menghapus data:
  1. Pertama, klik pada code produk (tulisan berwara orange)

  2. Maka tampil rincian data dari produk

  3. jika ingin menghapus* produk, pilih tombol "Delete"

    4. catatan : *produk tidak dapat dihapus jika sudah pernah mengalami transaksi.
Kategori Produk (Product Category)
Bagaimana menampilkan kategori produk?
    Menampilkan kategori produk
    Produk-produk yang ada dibagi/dipisah menjadi beberapa kategori.
    Untuk menampilkan kategori dari produk
  1. pilih menu master data → product → product master, seperti pada gambar

  2. Kemudian akan tampil data dari kategori produk



Bagaimana menambah/membuat kategori baru?
    Menambah/membuat kategori baru
  1. Untuk menambah kategori baru, pilih tombol (+) , dan akan tampil form untuk membuat kategori baru


  2. Isilah kolom yang ada sesuai dengan data yang di perlukan, kolom yang berwarna kuning artinya kolom tersebut wajib diisi, dan kolom yang berwarna putih tidak wajib diisi. Untuk menyimpan data pilih tombol "Save", atau tombol “Save&New” untuk menyimpan data dan langsung menampilkan form untuk mengisi data baru.

    3. Kategori baru untuk produk telah selesai dibuat.
Bagaimana mengedit kategori produk?
    Mengubah/mengedit produk kategori
    Untuk mengubah data dari kategori, maka :
  1. Pertama, klik pada name (tulisan berwara orange)

  2. Akan tampil rincian dari kategori.

  3. Isilah kolom, dengan data terbaru (data ter-update)
    Lalu pilih/klik tombol "Save" untuk menyimpan data terbaru.
Bagaimana menghapus kategori produk?
    Menghapus produk kategorik
    Untuk menghapus data dari kategori, maka :
  1. Pertama, klik pada name (tulisan berwara orange)

  2. Akan tampil rincian dari kategori.

  3. Kemudian pilih tombol "Delete"untuk menghapus kategori
    Kategori barang tidak akan dapat dihapus jika sudah pernah melakukan transaksi.

Daftar Harga (Price List)
Bagaimana menampilkan daftar harga?
    Menampilkan daftar harga
    Menampilkan daftar harga dari setiap produk dan harga jual untuk setiap produk.
  1. Pertama pilih menu master data → product → price list

  2. Dan tampil data dari price list


Bagaimana membuat daftar harga?
    Membuat daftar harga
    Untuk membuat daftar harga baru
  1. Pilih tombol (+) , untuk menampilkan form membuat daftar harga baru.

  2. Kemudian isi kolom, lalu klik tombol "Save" untuk menyimpan data, atau klik tombol "Save&Delete" untuk menyimpan dan menampilkan form baru.
Bagaimana mengubah daftar harga?

    Untuk mengubah data dari kategori, maka :
  1. Pertama, klik pada name (tulisan berwara orange)

  2. Akan tampil rincian dari kategori.

  3. Isilah kolom, dengan data terbaru (data ter-update)
    Lalu pilih/klik tombol "Save" untuk menyimpan data terbaru.
